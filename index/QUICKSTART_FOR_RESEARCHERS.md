# Quickstart for Researchers

If you are new to this repository, use this order.

## 15-minute route

1. `index/MASTER_INDEX.md`
2. `index/RESEARCH_TRAIL_MAP.md`
3. `index/BROWSE_BY_TOPIC.md`
4. `data/master/repo_master.csv`
5. `insights/adoption_backlog.md`

## 60-minute route

1. Cross-topic context:
   - `analysis/cross/deep_ai_dev_landscape_2026-03-09.md`
2. One technical lane:
   - pick one of `agent/mcp/rag/gateway/eval/security`
3. Risk intelligence:
   - `data/risks/failure-patterns.md`
   - `data/risks/upgrade-risk-matrix.md`
4. Decision output:
   - `insights/adoption_backlog.md`

## Quality checklist for any conclusion

- Includes source links
- Includes risk and boundary conditions
- Includes implication for OwlClaw-like architecture
- Includes what action should happen next
