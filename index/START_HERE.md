# Start Here

Owl GitHub Intel is a research publication: it converts scattered public AI ecosystem signals
(issues, PRs, releases, docs) into decision-grade artifacts: datasets, risk intelligence, and
adoption guidance.

This repository is independent. It was originally built to support OwlClaw, but it is not an
OwlClaw codebase or documentation site.

## What To Do (60 seconds)

1. Open the portal: `50-publish/site/`
2. Read the current decisions: `insights/adoption_backlog.md`
3. Inspect risks before adopting anything:
   - `data/risks/failure-patterns.md`
   - `data/risks/upgrade-risk-matrix.md`
4. If you need depth, pick a track under `analysis/` (agent/mcp/rag/gateway/eval/security).

## Core Outputs

- Master dataset: `data/master/repo_master.csv`
- Decisions: `insights/adoption_backlog.md`
- Risk intelligence:
  - `data/risks/failure-patterns.csv`
  - `data/risks/upgrade-risk-matrix.csv`
- Publish artifacts: `publish/digests/` and `publish/overview_public.md`
